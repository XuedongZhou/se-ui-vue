# 安装

### 使用包管理器

```shell
# NPM
$ npm install @se-ui/vue --save

# Yarn
$ yarn add @se-ui/vue

# pnpm
$ pnpm install @se-ui/vue
```

### 浏览器直接引入

```html
<head>
  <!-- Import style -->
  <link rel="stylesheet" href="//unpkg.com/@se-ui/vue/dist/index.css" />
  <!-- Import Vue 3 -->
  <script src="//unpkg.com/vue@3"></script>
  <!-- Import component library -->
  <script src="//unpkg.com/@se-ui/vue"></script>
</head>
```
